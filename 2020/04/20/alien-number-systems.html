Ten Fingers - Decimal Number system

    Since humans have been in existence, the need to track the amount of an something has been a necessity. A Number System is quite simply A set of values used to represent different quantities. The system we use as humans is the decimal number system which consists of ten digits from 0 to 9. These digits can be used to represent any numerical value. We can refer to decimal as a base 10 system since we must could 10 places before we move up in place value. This means that the number 10 in decimal would be 1 ‘ten’ plus 0 ‘units’. I’m going to assume we understand decimal arithmetic without having to explain it in detail(I hope).

    1’s and 0’s Binary Number System

    Unlike the decimal system, which is founded on base 10, the binary system is founded on base 2. 0 is represented as 0, and 1 is represented as 1 and just like the decimal system if the value exceeds 1 an additional column is needed. As binary is in base 2 it uses twos instead of tens which the decimal system uses. So for example 10 would translate to 8 plus 0 plus 2 plus 0 and would read 1010. The Zeros are present to show the columns 22 and 20 The first column on the far right is 2 to the power 0, for each subsequent column the power is increased by 1.

    Value 128’s 64’s 32’s 16’s 8’s 4’s 2’s 1’s
    Power 27 26 25 24 23 22 21 20

    Binary is used because it’s the easiest way to vary instructions through electric signals in a two-state system of on and off signals. On is represented as 1 and off as 0, though 0 is not actually no signal but signal at a lower voltage.

    Each binary digit is also called a bit. In any binary number, the rightmost digit is called least significant bit(LSB) and left most digit is called most significant bit(MSB).

    1 1 0 1 0
    MSB       LSB

    To find the decimal equivalent of any binary value we can find the sum of product of each digit with its positional value.

    110102 = 1 x 24 + 1 x 23 + 0 x 22 + 1 x 21 + 0 x 20

    = 16 + 8 + 0 + 2 + 0

    = 2610

    Hexadecimal Number System

    The Hexadecimal system has 16 digits - 0 to 9 and A to F where A is equal to 10, B is equal to 11 and so on till F. This system is also a positional value system with where each digit has its value expressed in powers of 16.
